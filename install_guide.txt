1. Open the terminal in your root directory of Sneat Laravel.
2. Use the following command to install the composer - composer install
3. Run the following command to generate the key - php artisan key:generate
4. By running the following command, you will be able to get all the dependencies in your node_modules folder: yarn
5. To run the project, you need to run the following command in the project directory. It will compile JavaScript and Styles. - yarn dev
6. To serve the application, you need to run the following command in the project directory - php artisan serve
